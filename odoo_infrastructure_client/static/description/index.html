Odoo Infrastructure Client

Remote Odoo Infrastructure Management


Odoo Infrastructure Client is an application developed by the Center of Research & Development company.

It allows you to manage remote Odoo infrastructure via RPC (remote procedure call).

With a little configuration, you can provide your customers with the final SaaS product.

Key features:
  • High-level server management
  • Pre-configured database templates
  • Automatic backups (very deep configured: backup every hour, daily, weekly, etc.)
  • Fully functional database management (create, drop, rename, duplicate, modules installation, etc.)
  • Frontend management (automatic generation of ssl certificates)
  • Customer portal (users can create, drop db, manage backups, etc.)
